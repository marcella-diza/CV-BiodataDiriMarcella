Implementasi Web Statis dengan Fungsi Interaktif JavaScript

Proyek ini menunjukkan pemahaman saya dalam menggabungkan tiga elemen inti front-end: struktur (HTML), gaya (CSS), dan logika interaktif (JavaScript). Fokus utamanya adalah membuat halaman yang terlihat baik dan memiliki fitur fungsional.

Tipe Proyek : Frontend Development (Interaktivitas)
Fungsi Utama : Kalkulator Sederhana & Validasi Formulir

Implementasi Fungsi JavaScript

Dalam proyek ini, saya berhasil mengimplementasikan dua fungsi inti JavaScript:

  • Kalkulator Sederhana: Menggunakan fungsi JS untuk mengambil input dari form, melakukan operasi matematika (+, -, *, /), dan menampilkan hasilnya ke elemen HTML tanpa me-refresh halaman (menggunakan manipulasi DOM).
  • Validasi Formulir: Membuat fungsi yang memastikan semua bidang input pada formulir kontak telah terisi dengan format yang benar sebelum data dapat disubmit. Ini meningkatkan User Experience.

Pelajaran yang Didapat

Proyek ini menguatkan pemahaman saya tentang:

DOM Manipulation

Cara JavaScript berinteraksi dengan struktur HTML (Document Object Model) untuk mengubah konten dan gaya secara dinamis.

Debugging Dasar

Mempelajari cara menggunakan `console.log()` dan Developer Tools di browser untuk melacak dan memperbaiki kesalahan kode JS.
